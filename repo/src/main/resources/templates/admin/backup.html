Backups

Nightly local backups (mysqldump + gzip), 14-day retention.

Recovery Procedure
  1. docker compose down
  2. Locate the most recent .sql.gz in /app/backups/ (or the backup-data volume)
  3. docker compose up mysql
  4. gunzip < backup.sql.gz | docker exec -i mysql mysql -uretail_user -pretail_pass retail_campaign
  5. docker compose up app

Estimated RTO: under 4 hours on a single on-prem server.
